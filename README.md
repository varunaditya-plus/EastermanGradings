# Easterman's Gradings

Just got your test results? Need some **praise** or **sadism** put on you? Get Easterman's opinion.

This is a simple website which plays Easterman's grading audios from The Outlast Trials when you input</br>
your test results. The order is randomised and played on a simple interface at [easterman.pages.dev](https://easterman.pages.dev/)
